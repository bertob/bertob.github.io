"Dynasty":http://tobiasbernard.com/dynasty is a visualization tool for succession timelines. I originally built it to make a timeline visualization of all the Roman Emperors, but I soon realized that with a few changes it would work for other data as well.
The result is a little tool that takes 2 CSV files, one for people on the timeline and one for the dynasties they’re part of, and draws a timeline of these people’s lives and reigns to an SVG element in the browser.

You can "try it for yourself":http://tobiasbernard.com/dynasty or get the code on "Github":https://github.com/bertob/dynasty.
